     1→# 图表制作工坊 - Worklog
     2→
     3→## Project Overview
     4→A rich, non-coder-friendly chart-making tool built with three JS libraries:
     5→- **ECharts 6** - data charts (bar/line/pie/radar/scatter/...)
     6→- **Mermaid 11** - flow & structural diagrams (flowchart/sequence/mindmap/gantt/...)
     7→- **AntV Infographic (@antv/infographic 0.2.19)** - declarative infographic visualization with 276 built-in templates and a text-based syntax
     8→
     9→## Architecture Decisions
    10→- All three editors live under `src/components/{echarts,mermaid,infographic}-editor/`
    11→- Shared shell in `src/components/chart-tool/` provides header (engine tabs), footer, save/load/AI dialogs
    12→- Charts persisted via Prisma `Chart` model (id, title, engine, type, config JSON, thumbnail)
    13→- Backend API at `/api/charts` (CRUD) and `/api/ai/suggest` (LLM-powered recommendation)
    14→- AI suggestions return ready-to-use config object so users can apply with one click
    15→- Each editor uses dynamic import (ssr: false) to keep vendor libs out of the server bundle
    16→- Live preview element is wrapped in a forwarded ref so the shell can snapshot thumbnails & PNG exports
